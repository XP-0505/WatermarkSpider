# 短视频平台视频图册去除水印

目前支持抖音、小红书、微博平台视频和图册去水印下载，其他平台正在开发，请耐心等待。。。

## 支持平台

| 平台 | 抖音 | 小红书 | 微博 | 网易云音乐MV |
|:--:|:--:|:---:|:--:|:-------:|  
| 图册 | √  |  √  | √  |    √    |
| 视频 | √  |  √  | √  |

## 请求示例

请求地址：localhost:8011/spider/specificInfo

请求方式：POST

传参格式：JSON

请求参数：

|  字段  |          说明           |
|:----:|:---------------------:|
| data |         分享的链接         |
| type | 类型（image-图册；video-视频） |

## 其他说明
网易云音乐MV无去水印视频
接口调用默认返回视频或图册下载链接地址，如要下载到本地，请自行调用Download工具类中的download方法即可。
